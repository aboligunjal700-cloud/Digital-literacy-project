# Project Overview
The Digital Literacy Project is designed to help students understand and effectively use digital tools in their academic and professional lives. 

As part of this project, I worked on five different tasks covering key areas of digital literacy. These include creating an infographic to spread awareness, building a professional online portfolio using platforms like GitHub and LinkedIn, exploring coding and collaboration tools, understanding professional email etiquette, and learning about cybercrime and online safety.
